##  HTML Formatting Elements


Formatting elements were designed to display special types of text:

 **Example:**

\<b> - Bold text\
\<strong> - Important text\
\<i> - Italic text\
\<em> - Emphasized text\
\<mark> - Marked text\
\<small> - Smaller text\
\<del> - Deleted text\
\<ins> - Inserted text\
\<sub> - Subscript text\
\<sup> - Superscript text
